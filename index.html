Some Of My Projects

  • Suro

    Solo Project
    React, Redux, Ruby on Rails, PostgreSQL, Google Maps API

    Suro is a pixel perfect clone of Turo, a car sharing marketplace. Users can list their cars to be rented or rent other users' cars. I learned about the frameworks many modern web apps are built on and put particular emphasis on styling and presentation.

    Repo Live Site

  • The Tellurian

    Group Project
    React, Redux, Express.js, Mongoose, MongoDB, Work Bank API, Various News APIs

    The Tellurian is an aggregate world news source where users can find and save breaking news stories from any country. I collaborated with 3 other engineers to construct a visual and scalable solution and learned about Git Workflow within a team.

    Repo Live Site

  • Color Crash

    Solo Project
    JavasScript Canvas API

    A side-scrolling game written in Javascript (with minimal libraries) and designed with physics emulation logic in mind. Animated using Canvas.js, players attempt to project a ball across a city in order to score based on distance traveled. I followed the directives of Object Oriented Programming when assembling the game logic.

    Repo Live Site
